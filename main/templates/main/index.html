Инструментарий для работы с концептуальными схемами в родоструктурной форме Web-exteor


Данный сайт предназначен для работы с концептуальными схемами в родоструктурной форме и может быть использован при обучении и совместной исследовательской работе. На данные момент функционал позволяет создавать и редактировать родоструктурные формы концептуальных схем, загружать РС-формы КС из специализированного ПО Экстеор, создавать и редактировать конституенты, проверять их статус и типизацию, предоставлять доступ к личным концептуальным схемам другим пользователям и др.
Сайт будет полезен для лиц, осваивающих математический аппарат языка родов структур, студентов высших учебных заведений, учащихся и других заинтересованных пользователей.
